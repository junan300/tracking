This app now uses a modern build system. To run it, please use one of the following methods:

Quick Start Options:

  • Web version: Open terminal and run npm run dev
  • Desktop app: Open terminal and run npm start
  • Simple launcher: Double-click launcher.bat

Why can't I just double-click index.html?

The app has been refactored to use Vite and React modules, which require a build step or development server. This makes the code more maintainable and allows for better performance optimizations.
